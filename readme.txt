== Description ==

Shows related posts on MultiLanguage sites, working with WPML

= Features =

- Shows related posts on sites, working with WPML, dependent on current language
- Has custom classes on ul, li, img, wrapper elements.
- Easy to use.

== Installation ==

1. Upload `related-posts-for-wpml` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Settings -> Related Posts for WPML 
4. For using please call the function rpf_related() on your single page template
5. Enjoy
